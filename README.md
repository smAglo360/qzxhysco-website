# Pixel Cat Portfolio

Welcome to **Pixel Cat Portfolio**, a cozy pixel-art-inspired portfolio website built around playful visuals, animated interactions, and a charming cat-themed aesthetic.

## What this is

This project is a static portfolio landing page for **Qzxhysco**. It showcases featured works, a photo gallery, and a bright, retro pixel UI with theme and language toggles.

## Key features

- Retro pixel-style design with bold colors and animated reveal effects
- Featured works gallery with clickable thumbnails
- Image viewer modal for gallery and project previews
- Light / dark theme toggle
- English and Russian text support
- Built with Parcel for fast local development and optimized builds

## Files you’ll find here

- `index.html` — main website structure
- `styles.css` — site styling and theme support
- `script.js` — interactive behavior and translations
- `assets/` — image assets used in the portfolio
- `dist/` — production build output
- `package.json` — project metadata and npm scripts
- `.gitignore` — ignored files and folders

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open the local URL shown in your terminal.

## Build for production

```bash
npm run build
```

The built site will be generated into the `dist/` directory.

## Notes

This repository is a static portfolio demonstration. It can be hosted on GitHub Pages or any static site host.
